Python

Python là ngôn ngữ lập trình thông dịch do Guido van Rossum tạo ra năm 1990. Python có cơ chế tạo kiểu động và cấp phát bộ nhớ động. Python được phát triển trong một dự án mã nguồn mở do tổ chức phi lợi nhuận Python Software Foundation quản lý.

Python có cú pháp đơn giản, dễ hiểu, rất thuận tiện cho việc phát triển ứng dụng. Python có nhiều thư viện chuẩn, đặt biệt là các thư viện khoa học và tính toán như: Numpy, SciPy.

Pandas

Pandas là một thư viện được viết cho ngôn ngữ lập trình Python để thao tác và sử lý dữ liệu. Pandas cung cấp cấu trúc dữ liệu để thao tác với các bản số và chuỗi thời gian.

Cùng với Matplotlib, Numpy và SciPy, Pandas là thư viện được sử dụng rộng rãi trong Python. Trong quá trình sử lý dữ liệu, Pandas là một thư viện cực hữu ích: từ chuyển đổi hay ánh xạ dữ liệu thô sang dạng dữ liệu mà chúng ta mong muốn, nhằm có thể phân tích dễ dàng hơn.

Jupyter

Jupyter notebook là một ứng dụng web mã nguồn mở cho phép bạn tạo và chia sẽ các tài liệu chứa mã nguồn, phương trình, hình ảnh và các van bản. Với Jupyter notebook bạn có thể làm sạch và chuyển đổi dữ liệu, mô phỏng hình học, thống kê dữ liệu, machine learning và nhiều ứng dụng khác.

Jupyter notebook hỗ trợ hơn 40 ngôn ngữ lập trình, bao gồm cả python, R, Scala, Julia.

Keras

Keras là một API hỗ trợ mạng neural cấp cao, được viết bằng Python và có khả năng chạy trên TensorFlow, CNTK, và Theano. Keras được thiết kế với mục đích đi từ ý tưởng đến kết tối thiểu nhất nên rất thuận tiện cho việc nghiên cứu.

Với Keras, có thể dễ dàng để tạo một mô hình machine learning đơn giản, hỗ trợ mạng neural tích chập và hồi qui, và có thể chạy trên CPU và GPU.

Flask

Flask là một micro web framework nổi tiếng được viết bằng Python. Flask đơn giảng, nhẹ rất thích hợp cho người mới bắt đầu phát triển web bằng Python.

Flask là open source, và được xây dựng dựa trên hai thư thư viện chính là Werkzeug và Jinja 2.
